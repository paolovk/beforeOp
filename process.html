Ett nav element är också ett semantiskt element som underlättar både för tolkning av kod och dessutom för användare med synnedsättningar som kan enkelt navigera i hemsidan. Den berättar att det finns länkar som navigerar användaren inom sidan eller externt. (Bailey. E, 2018)

Det är standarden att navigationsmenyer ska vara i en UL för det underlättar då man ska skapa undermenyer men det finns diskussioner att det kan vara bättre att använda div eller span då det ökar tillgängligheten för synskadade (Coyer, C, 2013)
